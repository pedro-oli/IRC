## Como rodar
Abra 2 terminais:

* No primeiro, digite:
```
make all
./server <Porta>
```
> Ex.: ./server 1234

* No segundo, digite:
```
./client localhost <Porta> <Apelido>
```
> Ex.: ./client localhost 1234 pedro
